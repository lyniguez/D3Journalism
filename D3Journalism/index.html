This scatter plot shows the correlation for each state between households that receive food stamps with children under the age if 18 and how many children are in the household. This interactive scatter plot has two x-axis options for a 3 child household and one for a 4 child household. The data value for 3 and 4 child households increases as the number of households in that state receive food stamps. Both 3 and 4 child households had greater than a 0.7 correlation.
